API

GET /api/programs — full program list as JSON

GET /api/changes — intel feed; free = delayed 24h + capped, Pro/Team = real-time. Auth: Authorization: Bearer bsk_… or x-api-key: bsk_…

POST /api/analyze — body: {code, program_id?, repo_url?} → finding classes (free: 5/day per IP)

GET /api/whoami — tier + quota for the presented key

POST /api/subscribe — body: {tier} → Stripe checkout URL (402)

POST /api/confirm — body: {session_id} → returns your api_key

GET /api/status — system health
